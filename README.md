# Акустик

Десктоп-приложение для расчёта уровней шума на строительных и промышленных площадках.

**Нормативная база:** СП 51.13330.2011, ГОСТ 31295.2-2005, СанПиН 1.2.3685-21

---

## Возможности

- Расчёт уровней шума от точечных и линейных источников
- Учёт экранов-барьеров (метод Маекавы)
- Затухание в воздухе, поглощение грунтом
- Сравнение с ПДУ по типу территории (жилая, промышленная, школа, больница)
- Графический редактор с загрузкой DXF-планов
- Построение изолиний уровней шума
- Экспорт данных в QGIS
- Генерация отчётов PDF / DOCX / JSON
- Интеграция с Claude Code через MCP

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/YuliyaSkryabina/Acoustic.git
cd Acoustic
```

### 2. Установить Python-зависимости

```bash
python -m pip install fastapi uvicorn pydantic aiosqlite matplotlib numpy mcp python-docx reportlab
```

### 3. Запустить API-сервер

```bash
python api/main.py
```

Сервер запустится на `http://127.0.0.1:8765`

> Проверка: откройте http://127.0.0.1:8765/docs — интерактивная документация API

### 4. Установить зависимости интерфейса

```bash
cd electron
npm install
```

### 5. Запустить приложение

```bash
npm run dev
```

Откроется окно приложения «Акустик».

## Использование

### Панель инструментов

| Кнопка | Действие |
|--------|----------|
| ↖ | Выбор объекта |
| ✋ | Перемещение вида (зажать и тянуть) |
| 🔴 | Добавить источник шума |
| 🔵 | Добавить расчётную точку |
| ▬ | Добавить экран-барьер |
| 📂 DXF | Загрузить план в формате DXF |
| ⚡ Рассчитать | Выполнить акустический расчёт |

### Пошаговый расчёт

**Шаг 1 — Загрузить план (необязательно)**

Нажмите 📂 DXF и выберите файл `.dxf`. План отобразится в виде фонового слоя.

**Шаг 2 — Разместить источники шума**

1. Нажмите кнопку 🔴 на тулбаре
2. Кликните на план в нужном месте
3. Заполните форму:
   - Описание (например: «Компрессор»)
   - Тип: точечный или линейный
   - Октавные уровни звуковой мощности Lw [дБ] для 8 полос (63–8000 Гц)
   - Координаты X, Y, Z
   - Режим работы: постоянный / непостоянный (с указанием времени работы)
4. Нажмите «Сохранить»

**Шаг 3 — Разместить расчётные точки**

1. Нажмите кнопку 🔵
2. Кликните на план (например, у фасада здания)
3. Заполните форму:
   - Описание (например: «Фасад дома №5»)
   - Тип территории: жилая, школа, больница, промышленная и др.
4. Нажмите «Сохранить»

**Шаг 4 — Добавить экраны (необязательно)**

1. Нажмите ▬
2. Кликните первую точку экрана → кликните вторую точку
3. Высота экрана задаётся в боковой панели

**Шаг 5 — Запустить расчёт**

Нажмите ⚡ **Рассчитать**. Результаты появятся:
- На плане: расчётные точки окрашиваются (зелёный = норма, красный = превышение)
- На плане: изолинии уровней шума
- В правой панели «Результаты»: таблица по октавам

### Таблица результатов

| Цвет строки | Значение |
|-------------|----------|
| Зелёный | Норма с запасом (Δ < -5 дБА) |
| Светло-зелёный | Норма (Δ < 0 дБА) |
| Жёлтый | Малое превышение (0–5 дБА) |
| Оранжевый | Среднее превышение (5–10 дБА) |
| Красный | Критическое превышение (> 10 дБА) |

### Нормативные уровни ПДУ

| Тип территории | День, дБА | Ночь, дБА |
|---------------|-----------|-----------|
| Жилая застройка | 55 | 45 |
| Школа / детский сад | 55 | 45 |
| Больница | 55 | 45 |
| Рекреационная зона | 55 | 45 |
| Промышленная | 80 | 80 |
| Офисная | 60 | 50 |

## Интеграция с Claude Code (MCP)

Добавьте в файл `.claude/mcp.json`:

```json
{
  "mcpServers": {
    "acoustik": {
      "url": "http://localhost:8765/mcp/sse",
      "transport": "sse"
    }
  }
}
```

Доступные инструменты:

| Инструмент | Описание |
|------------|----------|
| `calculate_noise` | Рассчитать уровни шума |
| `get_sources` | Получить список источников проекта |
| `add_source` | Добавить источник шума |
| `export_to_qgis` | Экспортировать данные в QGIS |
| `get_report` | Сформировать отчёт (json/docx/pdf) |

---

## Интеграция с QGIS

1. Скопировать папку `qgis_plugin/` в директорию плагинов QGIS:
   `C:\Users\<имя>\AppData\Roaming\QGIS\QGIS3\profiles\default\python\plugins\acoustik`
2. В QGIS: Модули → Управление модулями → найти «Acoustik» → включить
3. На панели появится кнопка «Загрузить данные Акустик»
4. Указать Project ID и нажать «Загрузить» — в QGIS создадутся слои источников и расчётных точек

## API документация

При запущенном сервере: http://127.0.0.1:8765/docs

Основные endpoints:

- `POST /calculate_noise/` — расчёт уровней шума
- `GET /sources/?project_id=...` — источники проекта
- `POST /export_to_qgis/` — экспорт GeoJSON
- `GET /report/?project_id=...&format=pdf` — отчёт

### Пример запроса

```bash
curl -X POST http://localhost:8765/calculate_noise/ \
  -H "Content-Type: application/json" \
  -d '{
    "sources": [{
      "id": "src1",
      "source_type": "point",
      "lw_octave": [85,85,85,85,85,85,85,85],
      "x": 0, "y": 0, "z": 1.0,
      "is_permanent": true,
      "duration_day": 16, "duration_night": 8,
      "directionality": 0
    }],
    "points": [{
      "id": "pt1",
      "x": 50, "y": 0, "z": 1.5,
      "territory_type": "residential"
    }]
  }'
```
